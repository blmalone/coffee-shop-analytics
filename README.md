# coffee-shop-analytics

Get started: 
```
clasp login # follow browser prompts to login

# Do this once
clasp clone <script_id>


# Do this at the start
clasp pull


# Do this to save
clasp push


# For versioning and deployments
clasp version "First version"

# Deployment
clasp deploy 1 "First deployment"

```
